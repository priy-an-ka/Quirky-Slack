# QuirkySlack

## Config

For local testing, create a `.env` file at the root of the directory with the following structure:
```
SLACK_SIGNING_SECRET=zzzzzzzz
SLACK_BOT_TOKEN=xxxxxxx
BOX_ID=yyyyy
WORDS_API_KEY=yyyyyyy
```
